### 📊 Sentiment Analysis on Social Media Posts

This project performs sentiment analysis on simulated social media posts to classify user opinions as Positive, Negative, or Neutral. It uses TF‑IDF for text feature extraction and Logistic Regression for classification, demonstrating a simple yet effective NLP workflow.

### 📌 Project Overview
Sentiment analysis helps identify emotions and opinions expressed in user‑generated content such as tweets, comments, and reviews.
This project builds a machine learning pipeline to:

Convert raw text into numerical features using TF‑IDF

Train a Logistic Regression classifier

Evaluate model performance using:

Precision

Recall

F1‑Score

Confusion Matrix
